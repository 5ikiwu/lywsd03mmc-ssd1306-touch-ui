# Notice

This repository contains documentation and prebuilt experimental firmware binaries for a modified Xiaomi LYWSD03MMC thermometer/hygrometer.

## Upstream Firmware

The firmware binaries are derived from the ATC firmware project:

- pvvx/ATC_MiThermometer: <https://github.com/pvvx/ATC_MiThermometer>
- atc1441/ATC_MiThermometer: <https://github.com/atc1441/ATC_MiThermometer>

The pvvx repository is MIT-licensed. This repository includes the MIT license text in `LICENSE`.

## Flasher

Flashing instructions use TelinkMiFlasher:

- <https://pvvx.github.io/ATC_MiThermometer/TelinkMiFlasher.html>

The flasher page credits atc1441 and pvvx.

## Visual Reference

The README links to a stock LYWSD03MMC reference image hosted by pvvx:

- <https://pvvx.github.io/ATC_MiThermometer/img/LYWSD03MMC.png>

The OLED UI mockup and wiring diagrams in this repository are original documentation assets for this project.
