# Photos

Place public build photos in this folder.

Recommended photos:

- Front view showing the OLED installed.
- Close-up of the OLED wiring pads.
- Close-up of the touch sensor wiring.
- Finished enclosure or mounting.
- A short sequence showing the different OLED pages.

Before committing photos:

- Remove EXIF metadata.
- Avoid showing MAC addresses, serial numbers, account names, mail labels, rooms, or other personal details.
- Use clear lighting and neutral backgrounds.
- Credit the photographer if the photo is not yours.

The main README currently uses a public stock LYWSD03MMC reference image hosted by pvvx and credits it in `NOTICE.md`.
